What is this repo ?

This is a repository made just for beginners who want to explore github and its features like pull request. When your pull request is merged you will be able to see your card in 'The Contributors' section (if done properly).

In this repository, you can request 1 or 2 pull request, since there is nothing more that can be done here. If you have signed up for Hacktober Fest , then you will only need 3 more pull requests to get a chance to win a t-shirt. This event is conducted so as to invite you to contribute to Open-Source Projects.

How to contribute ?

Go to this repository.

Open the index.html file.

Click on the edit icon on the top right (this will fork this repo into your account)

Check for the part where the comment says "Add your card above this and leave a line after it"

Paste the code found in the readme.md file, above this comment and make changes for your card.

After making suitable changes, make a pull request (Also do keep in mind the instructions in the readme file)
